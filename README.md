# Sales_-Dashboard_-Analysis
Project Overview
This project focuses on building an interactive sales dashboard using Power BI to analyze business performance across regions, product categories, and time periods. The dashboard helps transform raw data into meaningful insights for better decision-ma
# Objective
To visualize sales data and identify key trends, top-performing products, and regional performance using interactive dashboards.
#**Tools & Technologies**
* Power BI
* Microsoft Excel
* DAX (Data Analysis Expressions)
  📊 **Key Features**
* KPI Cards: Total Revenue, Total Orders, Total Quantity
* Monthly Sales Trend Analysis (Line Chart)
* Sales by Region (Bar Chart)
* Sales by Category (Donut Chart)
* Top 5 Products by Revenue
* Interactive Slicers (Region, Order Date)
# 📈 Key Insights
* Certain regions contribute higher sales compared to others
* Sales show clear monthly trends and fluctuations
* A few products generate the majority of revenue
* Category-wise distribution highlights top-performing segments
